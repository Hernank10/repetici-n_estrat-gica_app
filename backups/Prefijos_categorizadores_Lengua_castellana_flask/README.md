## 🌟 Descripción

Aplicación web educativa interactiva para el aprendizaje de **prefijos categorizadores del español**. Una herramienta completa que combina teoría, práctica gamificada y colaboración comunitaria para dominar la morfología española.

### ✨ Características Principales

- 📖 **Teoría Interactiva**: Explicaciones detalladas con ejemplos visuales
- 🎮 **Práctica Gamificada**: 100+ ejercicios con sistema de puntos, niveles y rachas
- 🃏 **Flashcards Inteligentes**: Sistema de repetición espaciada (algoritmo SM-2)
- 🏆 **Logros y Medallas**: 10 logros desbloqueables
- 👥 **Comunidad**: Comparte ejemplos y aprende de otros usuarios
- 📊 **Dashboard Personalizado**: Seguimiento detallado del progreso
- ⭐ **Sistema de Estrellas**: Evalúa tu precisión en cada práctica

## 🚀 Tecnologías Utilizadas

- **Backend**: Flask 2.3 (Python)
- **Base de Datos**: SQLite con SQLAlchemy
- **Frontend**: HTML5, CSS3, JavaScript (ES6)
- **UI Framework**: Bootstrap 5
- **Iconos**: Bootstrap Icons

## 📋 Requisitos Previos

- Python 3.8 o superior
- pip (gestor de paquetes de Python)
- Git
