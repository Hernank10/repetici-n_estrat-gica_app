# 📚 Heterónimos Academy

Plataforma educativa interactiva para el aprendizaje de heterónimos del español (pares léxicos con raíces diferentes como padre/madre, toro/vaca).

## ✨ Características

- 📖 **Teoría Gramatical y Semántica** - Explicaciones completas sobre heterónimos
- 🃏 **50+ Flashcards Interactivas** - Estudio con tarjetas de memoria
- ✍️ **Ejercicios de Completación** - Practica con oraciones contextuales
- 🎯 **Quiz de Opción Múltiple** - Pon a prueba tus conocimientos
- 📝 **Redacción Creativa** - Guarda tus propios ejemplos
- 📊 **Seguimiento de Progreso** - Estadísticas y logros

## 🚀 Tecnologías

- **Backend**: Flask (Python)
- **Frontend**: Bootstrap 5, HTML5, CSS3, JavaScript
- **Datos**: JSON (sin base de datos para facilitar despliegue)
